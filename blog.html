What is Codenspire?

Codenspire stands for the Code and Enspire. The Code describes instructions/ text written using a programming language by a computer program. The Enspire means to inspire. Codenspire is the blog site where I share knowledge, especially in Data & Software (Web Apps). Not only that, I really enjoy writing and sharing my understanding of data, software engineering, project management, product management, and information & communication technology (ICT). I hope it will inspire the readers.

  • Knowledge about technology stack. I would like to share my experiences.
